# Console - `LIST CLUSTERS`

Displays all configured clusters in the current database.

**Syntax:**

- Long Syntax:

  ```
  LIST CLUSTERS
  ```

- Short Syntax:

  ```
  CLUSTERS
  ```

**Example:**

- List current clusters on database:

  <pre>
  orientdb> <code class="lang-sql userinput">LIST CLUSTERS</code>

  CLUSTERS
