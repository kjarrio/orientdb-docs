Find the top 3 Countries from where Customers are from:

```sql
SELECT 
  Name as CountryName,
  in('IsFromCountry').size() as NumberOfCustomers 
FROM Countries
ORDER BY NumberOfCustomers DESC 
LIMIT 3
```
