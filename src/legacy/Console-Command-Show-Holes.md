# Displays the free holes in databases.

# Introduction

OrientDB keep the free space as "holes". To remove the holes you have to compact the database using the [compact database](SQLCompactDatabase.md) command.

# Syntax

```java
show holes
```

Example:

```java
orientdb> show holes
```
